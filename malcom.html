Malcolm X Quote

Description

A single-page HTML and CSS project centered around one of my favorite quotes by Malcolm X. This design-focused piece pairs powerful imagery—featuring Malcolm X and his speech in Washington—with a strong visual theme reflecting righteousness, conviction, and justice.

Key focuses:

  • Thoughtful use of color, contrast, and typography to match the tone of the quote
  • Strong thematic consistency to evoke emotion and purpose
  • Visual storytelling through layout and imagery
This project showcases my eye for design, theme development, and using visual elements to enhance meaning and message.

Tools/Resources Used

  • HTML
  • CSS
  • Adobe Photoshop
  • Google Fonts
