Evansville Elephants

Description

A multi-page front-end website designed for a fictional basketball team, the Evansville Elephants. Built using HTML, CSS, Bootstrap, and JavaScript, this project showcases modern, responsive web design with interactive features and a strong visual identity.

Key features:

  • Clean, responsive layout using Bootstrap for optimal viewing across devices
  • Custom JavaScript navigation menu for smooth and intuitive browsing
  • Interactive map highlighting key team locations and events
  • Multiple themed pages including Home, Players, Schedule, History, and About Us
  • Consistent branding and design elements for a cohesive user experience
This project highlights my front-end design skills, from layout and styling to interactive elements and user navigation, with an emphasis on both aesthetics and usability.

Tools/Resources Used

  • CSS
  • Javascript
  • Bootstrap
  • FontAwesome
  • Adobe Color
