Coffee House

Description

A front-end web design project where I recreated a modern coffee house website using HTML and CSS. This project was a hands-on exercise in layout design and visual styling, with a strong emphasis on responsive design principles.

Key focuses:

  • Learned and applied Flexbox to create structured, adaptable layouts
  • Practiced core web design principles including spacing, hierarchy, and consistency
  • Focused on replicating a clean, inviting aesthetic suitable for a coffee brand
This project helped strengthen my foundational skills in layout and visual design using only HTML and CSS.
